To start the simulation, press start and enter the temperature of your desire and press enter. Next enter your desired speed and press next. Then enter a Fuel level percentage ranging from 0% (empty) to 100% being full and press enter. The program will throw a warning if the fuel percentage you've entered is less than 10% gallon. After you press enter, the program will display the readings and display a warning if needed. It will also ask if you would like to continue and restart. Then you will need to input 0 for no or 1 for yes then enter. If you enter 1, the program will start from the beginning again. Note that there is no specific range set for the temprature and speed.
